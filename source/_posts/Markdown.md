This post is a beginner-friendly guide to writing Markdown. Markdown is a lightweight markup language that is easy to read and easy to publish.

## 1. Headings

Use `#` to create headings. The more `#` you add, the smaller the heading becomes.

```markdown
# Heading 1
## Heading 2
### Heading 3
```

## 2. Emphasis

You can make text bold or italic.

```markdown
*italic*
**bold**
***bold italic***
```

## 3. Lists

### Unordered list

```markdown
- Item 1
- Item 2
- Item 3
```

### Ordered list

```markdown
1. First item
2. Second item
3. Third item
```

## 4. Links and images

```markdown
[OpenAI](https://openai.com)
![Example image](https://via.placeholder.com/150)
```

## 5. Code

Use backticks for inline code:

```markdown
Use `printf("helloWorld");` in your code.
```

Use triple backticks for a code block:

```c
#include <stdio.h>

int main() {
    printf("helloWorld");
    return 0;
}
```

## 6. Blockquotes

```markdown
> This is a blockquote.
```

- Keep paragraphs short and clear.
